```shell
# 2 arguments
$ echo hello world
hello world

# 2 arguments
$ echo hello      world
hello world

# 1 argument
$ echo "hello      world"
hello      world
```

- without quotes, words separated by spaces are treated as individual arguments. This is true in general for bash commands
